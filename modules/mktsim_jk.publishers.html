async schedule_close(seconds_til_close=5)[source]

The schedule_close function is used to schedule the closing of a market. It takes in an optional parameter, seconds_til_close, which defaults to 5 if not specified. The function then sleeps for the number of seconds specified by this parameter and closes the market by setting its status attribute to False.

Schedule close of market by setting status to False for Market. Closes market by setting status to False

Parameters:

  • self – Represent the instance of the object itself

  • seconds_til_close – Set the amount of time until the market closes

Returns:

The close_market function

Used to set the callback function when a price tick is updated. The callback should take two arguments: current price tick and new price.

async start_publishing()[source]

The start_publishing function is the main function of this class. It creates a task to schedule the market close, and then it enters an infinite loop that waits for price changes on a symbol and prints them out in real time. The loop will only exit when the market closes.

Parameters:

self – Represent the instance of the class

Returns:

True when the market is closed

mktsim_jk.publishers.market_publisher.main()[source]

This is the main function of the MarketPublisher. It creates a MarketPublisher instance and starts publishing.

Without any hooks, this will simply start moving the market, and optionally printing to the screen.
